# 🛞 TireFlow

Sistema de gerenciamento de estoque e vendas de pneus com integração ao **WhatsApp**, desenvolvido com foco em agilidade, organização e escalabilidade.

## 💡 Sobre o projeto

O TireFlow surgiu após observar de perto a rotina de um atacado de pneus e perceber como tarefas simples, como consultar produtos ou registrar uma venda, acabavam consumindo mais tempo do que deveriam.

A ideia foi desenvolver uma solução que aproveitasse uma ferramenta já presente no dia a dia da equipe: o WhatsApp. Em vez de exigir um novo sistema ou treinamento complexo, o objetivo é permitir que as operações sejam realizadas de forma prática através de comandos simples, mantendo o estoque sempre atualizado e facilitando o acompanhamento das movimentações.

Além de resolver um problema real, este projeto também faz parte da minha evolução como desenvolvedor, aplicando conceitos de arquitetura de software, organização de código e desenvolvimento backend em um cenário próximo ao encontrado no mercado.

## ✨ Funcionalidades

* Consulta rápida de pneus
* Controle de estoque
* Registro de vendas
* Entrada de produtos
* Ajuste de estoque
* Relatórios de movimentação
* Notificações em tempo real para o gestor

## 🛠️ Tecnologias

* Node.js
* TypeScript
* WhatsApp Web.js
* dotenv
* tsx

> Novas tecnologias serão adicionadas conforme o projeto evoluir.

## 🚧 Status

O projeto está em desenvolvimento e está sendo construído de forma incremental, com cada funcionalidade implementada e validada antes da próxima etapa.

## 🎯 Objetivo

Criar uma solução moderna, simples de utilizar e preparada para crescer, reduzindo erros operacionais e facilitando o gerenciamento de estoque através da automação de processos.
